Dicas:

Comando ;Comentario

LDA #$01  ;Carrega o valor hexadecimal $01 no registrador A
INX       ;Incrementa o valor do registrador X
SEC       ;Setar o carry flag
STA $0200 ;Armazena o valor do registrador A no local de memória $0200.
TAX       ;Transfere o valor de A para o registrador X
TAY	  ;Transfere o valor de A para o registrador Y
TXA	  ;Transfere o valor de X para o registrador A
TXY       ;Nao funciona
TYA	  ;Transfere o valor de Y para o registrador A

$#01 = Linguagem Hexadecimal 
$0202 = Linguagem Hexadecimal > 514 em decimal
#01 = Valor numérico Literal

LDA #$05
LDX #$00      ; Inicializa o contador (X)
LOOP:
STA $0200,X   ; Armazena no endereço base + offset de X
INX           ; Incrementa X
CPX #$FF      ; Compara X com #$FF
BNE LOOP      ; Se X não for 4, volta pro LOOP

LDA #$80
STA $01  armazena o valor do registrador A no local de memória $01
ADC $01  adiciona o valor armazenado no local de memória $01 

Diferença : #$01 e ADC $01
#$01 = Adiciona o valor %01 ao registro
ADC $01 = Adiciona o valor armazenado no endereço da memória $01 ao registrador A

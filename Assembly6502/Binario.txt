120 em binário é 01111000
10 em binário é 00001010

Vamos somar bit a bit, da direita pra esquerda, e vê os "vai um" (carries):


  01111000  (120)
+ 00001010  (10)
----------

1. 0 + 0 = 0
2. 0 + 1 = 1
3. 0 + 0 = 0
4. 1 + 1 = 0 (e "vai um" pro próximo bit)
5. 1 (do carry) + 1 + 0 = 0 (e "vai um" pro próximo bit)
6. 1 (do carry) + 1 + 0 = 0 (e "vai um" pro próximo bit)
7. 1 (do carry) + 0 + 0 = 1
8. 0 + 0 = 0

Juntando tudo, o resultado é 10000010.

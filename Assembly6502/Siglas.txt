BCC $90
;Branch on Carry Clear
salta (pula) se o bit Carry estiver LIMPO (0)
CLC       ; Clear Carry (Carry = 0)
BCC label ; Pula para "label" porque Carry = 0

BCS $B0
;Branch on Carry Set
 salta (pula) se o bit Carry estiver SETADO (1)
SEC       ; Set Carry (Carry = 1)
BCS label ; Pula para "label" porque Carry = 1

BCC continua se o flag estiver limpo ( 0 )
